# TDD Demo
ctrl+shift+N  打开相关目录
ctrl+shift+T  跳转至方法的测试文件
ctrl+shift+F10  运行代码
ctrl+alt+L   格式化代码
shift+alt+ctrl+T 将常量进行提取，放到工具类Constants中
alt+insert 创建构造类和getter和setter方法
alt+shift + ↓ 下移代码
ctrl + alt + N 代码的逆向提取 
ctrl+alt+V 提取变量
ctrl+shift+alt+T 将参数提取为一个类  introduce parameter object  
ctrl+alt+M 提取代码
ctrl+shift+T 生成测试类
